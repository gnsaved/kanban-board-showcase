# Pro Kanban Board (Design 4)

A clean, professional Kanban board UI built with Vanilla JavaScript, Tailwind (CDN), and SortableJS.

## Features
- Drag & drop tasks across columns (To Do / In Progress / Completed)
- Reordering within a column
- Saves both status and order to LocalStorage
- Priority tags (Low / Medium / High)
- Simple modal task creation and delete
