# Personal Task Board (Kanban Style)

A clean Kanban board with three columns: **To Do → Doing → Done**.

## Features
- Drag & drop tasks between columns
- Reorder tasks inside a column
- Priority tags (Low / Medium / High)
- Add / Edit / Delete tasks
- Auto-save with LocalStorage (your board persists after refresh)

## Run
Just open `index.html` in your browser.

## Deploy (GitHub Pages)
1. Push this folder to GitHub
2. Repo Settings → Pages
3. Source: `main` branch, folder: `/root`
4. Save → open your Pages link

## Notes
- Uses native HTML5 Drag & Drop (no libraries)
- Works offline
